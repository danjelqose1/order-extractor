# PDF Editor Manual QA Checklist

1. Open the app and select `PDF Editor` from the sidebar.
2. Upload a searchable PDF by clicking `Upload`, then repeat with drag-and-drop.
3. Verify the page canvas renders and the page thumbnail list shows all pages.
4. Use `Prev`, `Next`, `Fit Width`, `+`, and `-` to confirm navigation and zoom.
5. Rotate a page left and right, delete a page, and drag thumbnails to reorder pages.
6. Add a new text box, edit it in the right properties panel, and export the PDF.
7. Switch to `Edit Text`, click selectable PDF text, replace it, and confirm the right panel lists the replacement.
8. Upload a PNG/JPEG signature or image overlay and verify it appears on export.
9. Draw a freehand annotation, add a highlight, and add a rectangle.
10. Export/download and compare the original PDF file with the edited copy; the original must remain unchanged.
11. Use `Send to Extraction` and confirm the Dashboard extraction flow receives the edited copy.
12. Upload a scanned/image-only PDF and confirm the no-text-layer message appears while manual text boxes still work.
13. On a mobile-width viewport, confirm the editor uses the bottom toolbar and controls remain reachable.
